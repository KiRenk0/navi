# 参考焓法工程计算流程（按现有代码实现）

下面的流程描述严格对应当前代码实现方式，便于你逐步对照运行路径与计算环节。

### 1. 从规格文件进入计算入口

第一步是通过规格文件构建求解器入口，并初始化气体模型、翼型几何、采样网格与运行目录。

- 入口类：WingLowFidelitySolver 的初始化流程  
  代码位置：[solver.py](file:///e:/参考焓法/src/ref_enthalpy_method/solver.py#L47-L104)
- 规格加载与解析：Case/Vehicle/Sampling  
  代码位置：[models.py](file:///e:/参考焓法/src/ref_enthalpy_method/specs/models.py)、[loader.py](file:///e:/参考焓法/src/ref_enthalpy_method/specs/loader.py)
- 气体模型与黏度：常比热 + Sutherland  
  代码位置：[thermo.py](file:///e:/参考焓法/src/ref_enthalpy_method/gas/thermo.py)、[transport.py](file:///e:/参考焓法/src/ref_enthalpy_method/gas/transport.py)
- 采样网格与翼型几何  
  代码位置：[grid.py](file:///e:/参考焓法/src/ref_enthalpy_method/sampling/grid.py)、[dat_airfoil.py](file:///e:/参考焓法/src/ref_enthalpy_method/geometry/dat_airfoil.py)

### 2. 计算自由来流与有效来流参数

给定马赫数与高度，计算 (p∞, ρ∞, T∞, v∞)，并据攻角与后掠角得到有效攻角和有效马赫数。

- 标准大气：ISA1976 或 USSA1976  
  代码位置：[isa1976.py](file:///e:/参考焓法/src/ref_enthalpy_method/atmosphere/isa1976.py)、[ussa1976.py](file:///e:/参考焓法/src/ref_enthalpy_method/atmosphere/ussa1976.py)
- 有效攻角/马赫数  
  代码位置：[edge_conditions.py](file:///e:/参考焓法/src/ref_enthalpy_method/aero/edge_conditions.py#L12-L28)

### 3. 迎风面外缘参数：Busemann → 边界层外缘

迎风面每个弦向点以翼型坡度求局部切线角，再用 Busemann 公式给出压力系数 Cp，进而推得外缘压力、密度、温度与 Ma_e。

- Busemann 压力系数  
  代码位置：[busemann.py](file:///e:/参考焓法/src/ref_enthalpy_method/aero/busemann.py)
- 外缘参数计算链路  
  代码位置：[edge_conditions.py](file:///e:/参考焓法/src/ref_enthalpy_method/aero/edge_conditions.py#L31-L87)
- 迎风面边界层外缘缓存  
  代码位置：[windward_cache.py](file:///e:/参考焓法/src/ref_enthalpy_method/aero/windward_cache.py#L52-L112)

### 4. 前缘热流：Kemp-Riddell + 圆柱修正 + 后掠/攻角修正

前缘热流通过修正 Kemp-Riddell 公式得到球驻点热流，再转换为等效圆柱驻点，并考虑后掠与攻角修正。

- 前缘热流公式  
  代码位置：[leading_edge.py](file:///e:/参考焓法/src/ref_enthalpy_method/heatflux/leading_edge.py#L16-L120)
- 在求解器中接入前缘热流  
  代码位置：[solver.py](file:///e:/参考焓法/src/ref_enthalpy_method/solver.py#L148-L187)

### 5. 迎风面气动加热：参考焓法分支计算

对每个弦向点，用参考焓法求 h*，再根据 Re_x 的区间切换层流/湍流/对数律分支，得到 q_a(x)。

- 参考焓公式与分支热流  
  代码位置：[windward.py](file:///e:/参考焓法/src/ref_enthalpy_method/heatflux/windward.py#L16-L147)
- 迎风面热流分布与转捩权重  
  代码位置：[windward_cache.py](file:///e:/参考焓法/src/ref_enthalpy_method/aero/windward_cache.py#L113-L194)
- 转捩雷诺数与混合权重  
  代码位置：[transition.py](file:///e:/参考焓法/src/ref_enthalpy_method/aero/transition.py)

### 6. 背风面平均热流：Re_ns 相关式

背风面通过 Re_ns 相关式得到 St，再用 q = ρ∞ v∞ St (h_s − h_w) 得到热流分布。

- Re_ns 与 St 相关  
  代码位置：[leeward.py](file:///e:/参考焓法/src/ref_enthalpy_method/heatflux/leeward.py#L1-L56)
- 背风面热流分布  
  代码位置：[leeward.py](file:///e:/参考焓法/src/ref_enthalpy_method/heatflux/leeward.py#L59-L103)
- 求解器调用位置  
  代码位置：[solver.py](file:///e:/参考焓法/src/ref_enthalpy_method/solver.py#L206-L238)

### 7. 壁温求解：辐射平衡或瞬态平衡

壁温由热平衡方程解出，巡航段为稳态辐射平衡，助推段可选瞬态显式时间推进。

- 稳态辐射平衡通用求解  
  代码位置：[wall_temperature.py](file:///e:/参考焓法/src/ref_enthalpy_method/thermal/wall_temperature.py#L35-L121)
- 迎风面稳态耦合求解  
  代码位置：[windward_equilibrium.py](file:///e:/参考焓法/src/ref_enthalpy_method/thermal/windward_equilibrium.py)
- 背风面稳态耦合求解  
  代码位置：[leeward_equilibrium.py](file:///e:/参考焓法/src/ref_enthalpy_method/thermal/leeward_equilibrium.py)
- 瞬态显式时间推进  
  代码位置：[transient.py](file:///e:/参考焓法/src/ref_enthalpy_method/thermal/transient.py)

### 8. 结果输出与可视化入口

求解完成后将结果写入 runs/<case> 目录，包含 fields.npz 与 summary.json，同时可调用绘图脚本复现实验图。

- 结果组织与保存  
  代码位置：[solver.py](file:///e:/参考焓法/src/ref_enthalpy_method/solver.py)
- 绘图脚本入口  
  代码位置：[plot_wing_surface_temperature.py](file:///e:/参考焓法/src/plot_wing_surface_temperature.py)、[run_doc_suite.py](file:///e:/参考焓法/src/run_doc_suite.py)
